Internal Functions

These are built-in functions you can use.

Common

Name Meaning
number ^ degree Involutes number in degree
sin(number) Returns sine of number
cos(number) Returns cosine of number
tan(number) Returns tangent of number
asin(number) Returns arcsine of number
acos(number) Returns arccosine of number
atan(number) Returns arctangent of number
abs(number) Returns modulus of number
sqrt(number) Returns square root of number
ln(number) Returns natural logarithm of number
exp(number) Returns exponential function of number
