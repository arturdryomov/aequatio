Internal Functions

These are built-in functions you can use.

Common

Name Meaning
number ^ degree Involutes number in degree
sin(number) Returns sine of number
cos(number) Returns cosine of number
tan(number) Returns tangent of number
abs(number) Gives modulus of number
sqrt(number) Calculates square root of number

Minimization

Name Meaning
uniform( f, a, b, N ) Uniform method
bisection( f, a, b, l ) Bisection method
dichotomy( f, a, b, ε, l ) Dichotomy method
golden_ratio( f, a, b, l ) Golden Ratio method
fibonacci( f, a, b, l, ε ) Fibonacci method
quadratic_interpolation (f, x, Δx, ε1, ε2) Quadratic Interpolation method
configuration( f, [x1, x2, …], ε, [Δ1, Δ2, …], λ, α, ) Configuration method
rosenbrock( f, [x1, x2, …], ε, α, β, [Δ1, Δ2, …], N ) Rosenbrock method
adaptive_random( f, [x1, x2, …], α, β, M, N, R ) Adaptive Random method
