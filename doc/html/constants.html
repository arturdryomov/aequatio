Constants

Brief

Aequatio allows using constants in calculations.

There are two types of constants: built-in and user-defined. You can use both of them by simply typing name of a constant.

Using

Built-in Constants

For example, we have built-in constant named pi and want to calculate circle square with radius 5.

We type the command:

← pi * (5 ^ 2)

And Aequatio gives us the solution:

→ pi × 5 ^ 2 = 78.5398

There is a list of built-in constants.

User-defined Constants

Using user-defined constants is the same as using built-in ones, except that you should previously define a constant.

There is a special command called const to define a constant. Here is its format:

const <constant name> = <constant value>

Let’s define a constant with name radius and value 5. We’ll peek an example with circle.

← const radius = 5

→ Constant declared: radius = 5

← pi * (radius ^ 2)

→ pi × radius ^ 2 = 78.5398.

Constant value can be a number, another constant or even expression; for instance, the following command is absolutely correct:

← const square = pi * (5 ^ 2)

→ Constant declared: square = 78.5398.
