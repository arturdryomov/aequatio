Systems of Linear Equations Solving

Cramer

cramer([ [b1, b2, …, bn], [a11, a12, …, a1n], [a21, a22, …, a2n], … [an1, an2, …, ann] ])

Solves the following system of equations with Cramer method:

  • a11·x1 + a12·x2 + … + a1n·xn = b1;
  • a21·x1 + a22·x2 + … + a2n·xn = b2;
  • …
  • an1·x1 + an2·x2 + … + ann·xn = bn.

Example

← cramer([[2, 4], [1, 1], [1, -1]])

→ cramer([[2, 4], [1, 1], [1, −1]]) = [3, −1]

This means, that x = 3, y = −1 is the solution of the following system:

  • x + y = 2;
  • x − y = 4.

Gauss

gauss([ [a11, a12, …, a1n, b1], [a21, a22, …, a2n, b2], …, [an1, an2, …, ann, bn] ] )

Solves the following system of equations with Gauss method:

  • a11·x1 + a12·x2 + … + a1n·xn = b1;
  • a21·x1 + a22·x2 + … + a2n·xn = b2;
  • …
  • an1·x1 + an2·x2 + … + ann·xn = bn.

Example

← gauss([[1, 1, 2], [1, -1, 4]])

→ gauss([[1, 1, 2], [1, −1, 4]]) = [3, −1]

This means, that x = 3, y = −1 is the solution of the following system:

  • x + y = 2;
  • x − y = 4.
