Functions

Brief

Aequatio provides using functions in your calculations.

There are two types of functions: internal and user-defined. You can use both of them simple typing name of function with arguments (usually in brackets).

Using

Internal

For example, we have internal function named as sin. I want to calculate sine of 90°.

I’ll type that command:

← sin(90)

And Aequatio’ll give me the answer:

→ 0.893996663600558

Problem solved.

There is a list of internal functions.

User-defined

Using user functions is the same as internal, but you must define function before using it. Of course, you need to define function with name, that is not using as internal.

For defining function there is special command named func. Here you can see it’s format:

func "function name"("arguments") = "operations with arguments"

Let’s define function with name radius. I’ll peek an example with circle square.

← func radius(x) = pi * (x ^ 2)

→ Function ‘radius’ is declared

← radius(5)

→ 78.539816339744831
