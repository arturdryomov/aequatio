Finding Minimum: Zero Order

One-dimensional minimization

These methods operate only one variable functions.

Uniform

uniform(f, [a, b], N)

  • f — function name;
  • [a, b] — search interval;
  • N — number of iterations.

Example

← func f(x) = 2 * (x ^ 2) - 12 * x

→ Function declared: f(x) = 2 × x ^ 2 − 12 × x

← uniform(f, [0, 10], 9)

→ uniform(f, [0, 10], 9) = 3

Bisection

bisection(f, [a, b], l)

  • f — function name;
  • [a, b] — search interval;
  • l > 0 — accuracy.

Example

← func f(x) = 2 * (x ^ 2) - 12 * x

→ Function declared: f(x) = 2 × x ^ 2 − 12 × x

← bisection(f, [0, 10], 1)

→ bisection(f, [0, 10], 1) = 3.125

Dichotomy

dichotomy(f, [a, b], l, ε)

  • f — function name;
  • [a, b] — search interval;
  • l > 0 — accuracy;
  • ε > 0 — small number.

Example

← func f(x) = 2 * (x ^ 2) - 12 * x

→ Function declared: f(x) = 2 × x ^ 2 − 12 × x

← dichotomy(f, [0, 10], 1, 0.2)

→ dichotomy(f, [0, 10], 1, 0.2) = 2.85625

Fibonacci

fibonacci(f, [a, b], l, ε)

  • f — function name;
  • [a, b] — search interval;
  • l > 0 — length of finish interval;
  • ε > 0 — difference constant.

Example

← func f(x) = 2 * (x ^ 2) - 12 * x

→ Function declared: f(x) = 2 × x ^ 2 − 12 × x

← fibonacci(f, [0, 10], 1, 0.01)

→ fibonacci(f, [0, 10], 1, 0.01) = 2.69731

Quadratic Interpolation

 quadratic_interpolation (f, x, Δx, ε1, ε2)
  • f — function name;
  • x — start point;
  • Δx > 0 — step size;
  • ε1, ε2 — small numbers, which set accuracy.

Example

← func f(x) = 2 * (x ^ 2) + 16 / x

→ Function declared: f(x) = 2 × x ^ 2 + 16 ÷ x

← quadratic_interpolation(f, 1, 1, 0.003, 0.03)

→ quadratic_interpolation(f, 1, 1, 0.003, 0.03) = 1.61214
