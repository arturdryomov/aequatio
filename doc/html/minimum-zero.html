Finding Minimum: Zero Order

One-dimensional minimization

These methods operate only one variable functions.

Uniform

uniform(f, a, b, N)

  • f — function name;
  • a, b — search interval;
  • N — number of iterations.

Example

← func f(x) = 2 * (x ^ 2) - 12 * x

→ Function declared: f(x) = 2 × x ^ 2 − 12 × x

← uniform(f, 0, 10, 9)

→ uniform(f, 0, 10, 9) = 3

Bisection

bisection(f, a, b, l)

  • f — function name;
  • a, b — search interval;
  • l > 0 — accuracy.

Example

← func f(x) = 2 * (x ^ 2) - 12 * x

→ Function declared: f(x) = 2 × x ^ 2 − 12 × x

← bisection(f, 0, 10, 1)

→ bisection(f, 0, 10, 1) = 3.125

Dichotomy

dichotomy(f, a, b, l, ε)

  • f — function name;
  • a, b — search interval;
  • l > 0 — accuracy;
  • ε > 0 — small number.

Quadratic Interpolation

 quadratic_interpolation (f, x, Δx, ε1, ε2)
  • f — function name;
  • x — start point;
  • Δx > 0 — step size;
  • ε1, ε2 — small numbers, which set accuracy.

→ quadratic_interpolation(f, 1, 1, 0.003, 0.03) = 1.61214

Common minimization

Configuration

 configuration( f, [x1, x2, …], ε, [Δ1, Δ2, …], λ, α, )
  • f — function name;
  • [x1, x2, …] — start point, number of coordinates in number of function arguments;
  • ε > 0 — stop value;
  • [Δ1, Δ2, …] — steps on directions, number of directions is number of function arguments;
  • λ > 0 — acceleration coefficient;
  • α > 1 — decrease step coefficient.

Example

← func f(x, y) = 4 * ( (x - 5) ^ 2 ) + (y - 6) ^ 2

→ Function declared: f(x, y) = 4 × (x − 5) ^ 2 + (y − 6) ^ 2

← configuration(f, [8, 9], 0.3, [1, 2], 1, 2)

→ configuration(f, [8, 9], 0.3, [1, 2], 1, 2) = [5, 6]

Rosenbrock

 rosenbrock( f, [x1, x2, …], ε, α, β, [Δ1, Δ2, …], N )
  • f — function name;
  • [x1, x2, …] — start point, number of coordinates in number of function arguments;
  • ε > 0 — stop value;
  • α > 1 — acceleration coefficient;
  • -1 < β < 0 — decrease coefficient;
  • [Δ1, Δ2, …] — steps on directions, number of directions is number of function arguments;
  • N — maximum number of fail steps.

Example

← func f(x, y) = 4 * ( (x - 5) ^ 2 ) + (y - 6) ^ 2

→ Function declared: f(x, y) = 4 × (x − 5) ^ 2 + (y − 6) ^ 2

← rosenbrock(f, [8, 9], 0.6, 2, -0.5, [1, 2], 3)

→ rosenbrock(f, [8, 9], 0.6, 2, −0.5, [1, 2], 3) = [4.87963, 6.3254]

Random minimization

Adaptive Random

 adaptive_random( f, [x1, x2, …], α, β, M, N, R )
  • f — function name;
  • [x1, x2, …] — start point, number of coordinates in number of function arguments;
  • α > 1 — acceleration coefficient;
  • 0 < β < 1 — decrease coefficient;
  • M — number of fail tests;
  • N — maximum number of iterations;
  • R — minimum step size.

Example

← func f(x, y) = 4 * ( (x - 5) ^ 2 ) + (y - 6) ^ 2

→ Function declared: f(x, y) = 4 × (x − 5) ^ 2 + (y − 6) ^ 2

← adaptive_random(f, [8, 9], 1.618, 0.618, 3, 10, 0.8)

→ adaptive_random(f, [8, 9], 1.618, 0.618, 3, 10, 0.8) = [4.81129, 6.39804]

 returning_random( f, [x1, x2, …], β, M, N, R )

  • 0 < β < 1 — decrease coefficient;
  • M — number of fail tests;
  • N — maximum number of iterations;

Example
